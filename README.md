# Cadreon Sales Analysis

## Purpose

This analysis is to study the relationships between the sales of various products at Cadreon.

## Usage

This analysis uses the following libraries, which are all avaliable from CRAN:

```{r}
library(readxl)
library(tidyverse)
library(GGally)
```
